You can add file openapps.bat or openapps.command to run this code comfortably

I will tell you how to do

In Mac
1. Make a .command file
2. Write this code:
#!/bin/bash
python3 /your/full/file/path/filename.py
3.Rewrite to your file path and filename

And Done!

In window
1. Open notepad
2. Write this code:
@echo off
cls
echo Running Python Script...
python "C:/your/file/path/filename.py"
pause
3.Rewrite to your file path and filename
4.save file to .bat
